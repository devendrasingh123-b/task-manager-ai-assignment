# Task Manager – AI Assisted Full Stack Assignment

This project is a simple full stack Task Manager application built using **React (Frontend)** and **Node.js + Express (Backend)**.  
The assignment focuses on using **AI-assisted development tools** such as Copilot or Cursor while building the application.

---

## 🚀 Features

- Add new tasks
- View all tasks
- Mark tasks as complete / incomplete
- Delete tasks
- Filter tasks (All / Active / Completed)
- REST API with CRUD operations
- In-memory data store (no database)

---

## 🛠️ Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Node.js
- Express
- CORS

Tools:
- AI-assisted development (Copilot / Cursor)
